Домашнее задание
Задача
Расширить Weather API и добавить новые возможности.

🔹 Основное задание
Добавить endpoint:
GET /weather/{city}

Возвращает погоду по названию города.

Дополнить ответ рекомендацией, что надеть в зависимости от температуры:
холодно — тёплая одежда
прохладно — куртка
тепло — лёгкая одежда

Создать endpoint:
GET /weather/country/{country}

Возвращает информацию о погоде по городам страны.

Добавить аналитический endpoint:
GET /weather/country/{country}/top

Возвращает топ-3 самых тёплых городов в стране.
Требования:
отсортировать города по температуре
вернуть только 3 города с самой высокой температурой

🔹 Требования
использовать net/http и chi
ответ в формате JSON
разделение на слои: handler / service / client
обработка ошибок
использовать общедоступные API

🔹 Пример запроса
curl http://localhost:8080/weather/Almaty
