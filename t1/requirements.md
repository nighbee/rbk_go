# Проект: Текстовый процессор

## Введение
В этом проекте вы напишете CLI-инструмент на Go, который обрабатывает текстовые файлы согласно заранее заданным правилам.
Этот проект направлен на закрепление следующих навыков:
- Работа с файловой системой на Go
- Обработка и трансформация строк
- Конвертация чисел между системами счисления
- Работа с аргументами командной строки

## Требования
- Язык реализации: Go
- Использовать исключительно стандартные пакеты
- Соблюдайте best practices по Go-разработке
- Структура проекта должна быть чистой и масштабируемой

## Формат вызова
Программа получает два аргумента:
1. Имя входного файла с текстом
2. Имя выходного файла, в который будет записан результат

```bash
go run . input.txt output.txt
```

## Правила обработки
### 1. Конвертация по шаблонам:
| Шаблон | Действие |
|---------|----------|
| (hex) | Заменяет предыдущее слово, считая его шестнадцатеричным числом, на десятичное |
| (bin) | Заменяет предыдущее слово, считая его двоичным числом, на десятичное |
| (up) | Преобразует предыдущее слово в ВЕРХНИЙ РЕГИСТР |
| (low) | Преобразует предыдущее слово в нижний регистр |
| (cap) | Преобразует предыдущее слово так, чтобы первая буква была заглавной |
| (up, N), (low, N), (cap, N) | Применяет соответствующее преобразование к N предыдущим словам |

### 2. Пунктуация:
- ., ,, !, ?, : и ; должны стоять вплотную к предыдущему слову и отделяться пробелом от следующего.
- Исключение: последовательности вроде ..., !? сохраняются без изменений.
- Кавычки ' ставятся вплотную к одному слову или фразе внутри них (без пробелов).

### 3. Артикли:
- Артикль "a" заменяется на "an", если следующее слово начинается с гласной ([a, e, i, o, u]) или h.

## Пример использования:
```bash
echo "it (cap) was the best of times, it was the worst of times (up) , it was the age of wisdom..." > sample.txt
go run . sample.txt result.txt
def cat result.txt"
it the best of times, it was the worst of TIMES, it was the age of wisdom,
it Was The Age Of Foolishness,...
```
